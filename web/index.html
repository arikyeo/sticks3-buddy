StickS3 Buddy

Flash the M5StickC Plus S3 desk-pet firmware straight from your browser — no PlatformIO, no toolchain.

Your browser can't talk to serial devices. Open this page in desktop Chrome, Edge, or Opera.

Before you start

  1. Use desktop Chrome, Edge, or Opera (Web Serial is required).
  2. Connect the StickS3 to your computer over USB-C.
  3. Put it in download mode: long-press the side power button for 6 seconds to power it fully off, then plug/replug USB while holding the BOOT button until the green LED flashes.
  4. Click Install firmware and pick the device's serial port.
  5. Choose Install and wait for the flash to finish.

No driver install needed on Windows 11 — the StickS3 uses native USB-CDC, which Windows 11 handles out of the box. Older Windows versions may need a CDC driver.

The COM port changes between modes. The bootloader (download mode, used for flashing) and the running app (normal mode) enumerate as different USB-CDC ports. If the installer can't find your device after a flash finishes and it reboots, re-pick the port — it likely moved.

After flashing

Pair the buddy with Claude Code or Codex by installing the bridge and connecting over BLE. See the README for pairing steps and release notes for what changed.
